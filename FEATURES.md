# Features

## Authentication & Onboarding
- Email/password registration and login
- Phone number authentication (SMS OTP)
- Social login (Google, Apple, GitHub)
- Splash screen with app branding
- Multi-step onboarding screens (interests, avatar, notifications opt-in)
- Password reset / forgot password flow
- Session management (multi-device, logout everywhere)

## User Profiles
- Display name, username (@handle), and bio
- Profile avatar with upload/crop
- Cover photo
- Verification badge (color: `#c15f3c`)
- Follower / following counts and lists
- Profile links (website, social handles)
- Activity status (online / offline / last seen)

## Social Feed
- Text posts with optional media (up to 10 images/videos per post)
- Multiple reaction types (like, love, haha, wow, sad, angry)
- Comments with text and media support
- Repost (quote repost with added text)
- Polls (multiple choice, vote counts, expiration)
- Hashtag detection and linking
- Bookmark posts for later
- Post visibility controls (public / followers only)
- Delete and edit post (within grace period)

## Feeds
- **For You** – algorithmic feed based on engagement and interests
- **Following** – chronological feed from people you follow
- **Explore** – trending posts, hashtags, and suggested users
- Search posts, users, and hashtags

## Messaging
- Direct messages (1-on-1)
- Group chats (up to 250 members)
- Channels (broadcast, subscribers can't send)
- Full media sharing (images, video, audio, files)
- Typing indicators
- Read receipts (delivered / read)
- Message editing and deletion
- Reply threads within chats
- Forward messages
- End-to-end encryption (E2EE) for DMs and groups
- Mute / archive conversations

## Voice Calls
- One-on-one voice calls via WebRTC or third-party service
- Call notifications and ringing
- Mute / speaker / end controls

## Stories
- Photo and video stories (24-hour ephemeral)
- Story viewer list (who viewed)
- Reply to stories via DM
- Text and sticker overlay on stories

## Moderation
- Block users
- Mute users (posts, stories, messages)
- Report content (post, comment, message, profile)
- Content filtering (auto-moderation for hate speech, spam, NSFW)
- Admin panel (user bans, content takedown, report queue)

## Notifications
- Push notifications (FCM / APNs)
- In-app notification feed
- Notification preferences (per-type toggle)
- Types: like, comment, repost, follow, message, call, story reply, mention

## Monetization
- Banner ads in feed
- Sponsored posts (paid promotion)
- Badge / verification purchase (future)

## Account Management
- Edit profile (display name, bio, avatar, cover photo)
- Change email / phone
- Delete account with confirmation
- Privacy settings (who can see posts, send messages, view online status)
- Notification settings
- Blocked and muted user lists
- Account data export
